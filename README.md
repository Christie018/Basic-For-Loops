# Lab Directions

Description
Prompt the user to enter a number.
Use a for-loop to calculate the sum of all the numbers from 1 to the number entered.

## Example

>> Enter a number: << 100
>> 5050
Would you like to continue (y/n)? << y
>> Enter a number: << 20
>> 210
Would you like to continue (y/n)? << n
>> Goodbye!
